# Resume Website

Personal resume website for Rahul Panchal — a clean, responsive single-page site built with HTML, CSS, and JavaScript.

**Live site:** [rahulpanchal8.github.io/resume](https://rahulpanchal8.github.io/resume)

## Local development

Open `index.html` in a browser, or serve locally:

```bash
python3 -m http.server 8080
```

Then visit [http://localhost:8080](http://localhost:8080).

## Customize

Edit `index.html` to update your experience, skills, projects, and contact details.

## Deploy

This repo uses GitHub Actions to deploy to GitHub Pages on every push to `main`.
